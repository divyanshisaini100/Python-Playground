# ✅ Common String Check Methods:

Method	What it checks for

**isupper()**	All characters are uppercase (and at least one letter)
**islower()**	All characters are lowercase (and at least one letter)
**isalpha()**	Only alphabetic characters (no digits, no spaces)
**isdigit()**	Only digits (0–9)
**isalnum()**	Only alphanumeric (letters + digits)
**isspace()**	Only whitespace characters (space, tab, newline, etc.)
**istitle()**	String is in title case (e.g., "Hello World")
**isdecimal()**	Only decimal characters (mostly same as isdigit())
**isnumeric()**	Numeric (includes superscript numbers, etc.)
**isidentifier()**	Valid Python identifier (e.g., "var1")
**isprintable()**	All characters are printable (no escape/newline)
